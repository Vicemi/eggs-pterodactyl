# 🥚 eggs-pterodactyl

Repositorio de **eggs personalizados para Pterodactyl** desarrollados por mí.

🌐 Sitio web: https://vicemi.dev

---

## 📦 Sobre este repositorio

Este repositorio contiene configuraciones (*eggs*) listas para importar en el panel de **Pterodactyl**, pensadas para facilitar el despliegue de distintos servicios y aplicaciones.

Cada egg incluye:
- Configuración optimizada
- Variables predefinidas
- Scripts de instalación
- Compatibilidad con entornos modernos

---

|--------|------------|--------|

---

## 🚀 Uso

1. Descargar el archivo `.json` del egg deseado.
2. Ir al panel de administración de Pterodactyl.
3. Importar el egg en la sección correspondiente.
4. Crear un servidor usando ese egg.

---

## 🛠️ Requisitos

- Panel Pterodactyl instalado
- Wings configurado correctamente
- Permisos de administrador para importar eggs

---

## 📌 Objetivo

Centralizar y compartir eggs personalizados, manteniendo una estructura clara y reutilizable para distintos proyectos.
